# 電卓アプリ (アーカイブ)

このディレクトリには、以前 `main` ブランチのルートに公開されていた電卓 Web アプリのコードが保管されています。
新しい Web アプリ (国土交通省ニュースウォッチャー) への置き換えに伴い、ここに退避しました。

## 動作確認

```bash
# このディレクトリで簡易サーバーを起動
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

## ファイル

- `index.html` — 電卓 UI
- `styles.css` — スタイル
- `script.js` — 電卓ロジック
